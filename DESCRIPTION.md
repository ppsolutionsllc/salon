# Описание проекта Aesthetic Prime (простыми словами)

## Что это
Aesthetic Prime — это система для управления салоном красоты и сетью салонов.
Она помогает вести клиентов, сотрудников, услуги, записи и базовую аналитику в одном месте.

## Для кого система
- Для руководителя сети салонов
- Для администратора салона
- Для оператора
- Для мастера
- Для клиента

## Главная идея
Одна система обслуживает сразу несколько салонов.
Каждый пользователь видит только те салоны, к которым у него есть доступ.

## Основные возможности
- Ведение списка салонов
- Ведение базы клиентов
- Ведение базы сотрудников/мастеров
- Ведение каталога услуг (цена, длительность, буферы времени)
- Календарь записей
- Статусы записей и история изменений
- Базовые отчёты и аналитика
- Шаблоны сообщений (SMS/Email)
- Файлы (публичные и внутренние)

## Как работает запись клиента
1. Выбирается салон, услуга, мастер, дата и время.
2. Система проверяет, свободен ли мастер.
3. Создаётся запись со статусом `NEW` (новая).
4. Формируется код подтверждения (OTP).
5. После подтверждения кодом запись становится `CONFIRMED`.
6. Далее запись можно завершить, отменить, перенести или отметить неявку.
7. Все изменения статуса сохраняются в истории.

## Роли и их задачи

### CRM (админ/оператор)
- Управление салонами
- Управление клиентами
- Управление персоналом
- Управление услугами
- Работа с календарём записей
- Просмотр отчётов и аналитики
- Управление пользователями и правами

### Мастер
- Просмотр своего расписания
- Подтверждение и завершение записей
- Отметка неявки клиента

### Клиент
- Пошаговая запись на услугу
- Подтверждение записи кодом
- Просмотр своих записей
- Просмотр профиля

## Важные детали текущего состояния
- Архитектура готова для сети салонов (multi-salon).
- Разделение прав доступа уже реализовано.
- Есть фоновая обработка событий (например, для уведомлений).
- Часть разделов (публичное бронирование, некоторые метрики и интеграции) пока реализована частично или в виде заготовок.

## Результат для бизнеса
Система упорядочивает ежедневную работу салона:
меньше ручной рутины, понятный календарь, прозрачная база клиентов и контроль процессов по каждому салону в сети.
